## 5. 테스트 가이드

### 5.1 리사이저블 패널 테스트

1. 사이드바와 메인 콘텐츠 사이의 경계를 드래그
2. 크기가 조정되는지 확인
3. 새로고침 후 크기가 유지되는지 확인

### 5.2 탭 시스템 테스트

1. 메뉴 클릭 시 새 탭 열림 확인
2. 탭 클릭으로 전환 확인
3. Ctrl+Tab 단축키 동작 확인
4. X 버튼으로 탭 닫기 확인

### 5.3 커맨드 팔레트 테스트

1. Ctrl+Shift+P 누르기
2. 검색어 입력 후 필터링 확인
3. Enter로 메뉴 이동 확인
4. Escape로 닫기 확인

### 5.4 AI 패널 테스트

1. AI 버튼 클릭으로 패널 열기
2. 메시지 입력 후 전송
3. AI 응답 표시 확인
4. Ctrl+Shift+I 단축키 확인

## 6. 용어 사전

| 용어 | 설명 |
|------|------|
| **ResizablePanel** | 드래그로 크기를 조정할 수 있는 영역 |
| **PanelResizeHandle** | 패널 사이의 드래그 핸들 |
| **CommandPalette** | 명령어를 검색하여 실행하는 UI |
| **Zustand** | React용 경량 상태 관리 라이브러리 |
| **cmdk** | 커맨드 팔레트를 위한 라이브러리 |
| **FAB** | Floating Action Button, 화면에 떠 있는 버튼 |
